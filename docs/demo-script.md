# Demo Script

## Objective

Show a stable 60 to 90 second workflow from benchmark mutation data to explainable research output.

## Opening

"VaxAgent is an explainable oncology research workflow copilot. This is a research-use prototype, not a clinical product. The goal is to help a team interpret mutation data, prioritize neoantigen candidates, and communicate the output clearly."

## Happy Path

1. Start on the single-page dashboard.
2. Point to the benchmark dataset badge and note that the flow is deterministic for demo stability.
3. Briefly read the mutation summary cards.
4. Scroll to the ranked neoantigen candidates and call out the top-ranked entry.
5. Click that candidate and read the plain-English explanation panel.
6. Move to the mRNA blueprint preview and explain that it is a draft construct preview for research discussion, not a production-ready design.
7. Click the export action and mention that the report packages the same summary for sharing.
8. Close with the core value proposition: clearer interpretation, faster prioritization, and better communication across mixed research stakeholders.

## Talking Points

- one benchmark dataset only
- one stable UI flow
- deterministic outputs over fragile live integrations
- explanations are translated into plain English
- human review remains required

## Limitations To Say Out Loud

- the candidate ranking is a benchmark fixture for demo purposes
- the blueprint preview is a research draft, not an optimized therapeutic construct
- the app does not make treatment recommendations
- no real patient uploads are supported in this MVP
